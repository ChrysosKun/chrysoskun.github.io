categories: [linux]
comments: true
---

| 快捷键 | 作用                                              |
| :----: | :------------------------------------------------ |
| ctrl-r | 搜索命令行历史记录,重复按下ctrl-r会向后查找匹配项 |
| ctrl-a | 将光标移至行首                                    |
| ctrl-e | 将光标移至行尾                                    |
| alt-b  | 以单词为单位移动光标(向前)                        |
| alt-f  | 以单词为单位移动光标(向后)                        |
| ctrl-w | 删除键入的最后一个单词                            |
| ctrl-u | 删除行内光标所在位置之前的内容                    |
| ctrl-k | 删除光标至行尾的所有内容                          |
| ctrl-l | 清屏                                              |
